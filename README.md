# odin-tic-tac-toe game
A tic-tac-toe game made for The Odin Project, utilizing vanilla HTML, CSS, and JS.
## Game Rules
- Players use a form to enter their names and take turns marking up a 3x3 grid of squares.
- Wins are classified as 3 marks from the same player in either a horizontal row, a vertical row, or a diagonal.
- A tie occurs if the grid is completely filled but no wins have taken place.
- To play again, simply fill out and submit the form again.
## Local Setup
1. Clone the repo (SSH key): ```git@github.com:curiousConductor49/odin-tic-tac-toe.git```
2. Use VSCode's live preview extension to run it locally in the browser!
